画像を編集して表示するサンプルです。

## サンプルについて 画像ファイルを選択するとCloudinaryに画像がアップロードされ、加工された画像が表示されます。 ------------------ ### 必要なAdd-on * [Cloudinary](https://addons.heroku.com/cloudinary) ------------------ ### 説明 画像ファイルを選択するとページ表示時に取得するCloudinaryへのアクセス情報を使ってブラウザから直接Cloudinaryに直接画像をアップロードします アップロードした画像はCloudinaryに渡すパラメータにより加工された画像として取得されます ------------------ ### 関連ソース(ソースの閲覧には権限が必要です。) - [ImageServlet.java](https://github.com/FLECT-DEV-TEAM/heroku-samples/blob/master/src/main/java/herokuSamples/web/image/ImageServlet.java) - [image.html](https://github.com/FLECT-DEV-TEAM/heroku-samples/blob/master/src/main/webapp/WEB-INF/image/image.html)
