アップロードした画像を編集して表示するサンプルです。
## サンプルの説明 画像ファイルを選択するとCloudinaryに画像がアップロードされ、加工された３種類の画像が表示されます。 * 200x200 * 100x100 * 150x150 円形 顔認識 アップロードされたファイルは１日１回、深夜のバッチ処理で削除されます。 ## 解説 サンプルはAddonのCloudinaryを使用してアップロードした画像を変換しています。

このサンプルを実行するには[Cloudinary](https://addons.heroku.com/cloudinary)アドオンが必要です

Addonの追加コマンドは以下です。(Starterは無料)

``` heroku addons:add cloudinary ```

Addonを追加すると環境変数に**CLOUDINARY_URL**という環境変数が設定される
### ダイレクトアップロード 画像ファイルを選択するとページ表示時に取得するCloudinaryへのアクセス情報を使ってブラウザから直接Cloudinaryに直接画像をアップロードします アップロードした画像はCloudinaryに渡すパラメータにより加工された画像として取得されます ### 関連ソース(ソースの閲覧には権限が必要です。) - [ImageServlet.java](https://github.com/FLECT-DEV-TEAM/heroku-samples/blob/master/src/main/java/herokuSamples/web/image/ImageServlet.java) - [image.html](https://github.com/FLECT-DEV-TEAM/heroku-samples/blob/master/src/main/webapp/WEB-INF/image/image.html)
