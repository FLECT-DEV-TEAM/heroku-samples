Sessionに値を保存するサンプルです。

セッションに保存する値を入力してください。

セッションに保存されている値は

"{{value}}"

です。

## サンプルについて ### 必要なAdd-on - [Memcachier](https://addons.heroku.com/memcachier) ### 関連ソース(ソースの閲覧には権限が必要です。) - [SessionServlet.java](https://github.com/FLECT-DEV-TEAM/heroku-samples/blob/master/src/main/java/herokuSamples/web/session/SessionServlet.java)
